## Unititled Quiz App

Quiz app website using a Laravel API and React frontend.

---

### Application Features

-   Login
-   Register
-   View Dashboard
    -   Most popular quizzes global
    -   Followed users
    -   Followed users quizzes
-   View Profile
-   Users quizzes
-   Users scores
-   Update profile
-   Follow user/profile

-   Explore

    -   Text search for quizzes
    -   Tag search for quizzes
    -   View results

-   Create Quiz
-   Details
-   Scores
-   Edit Quiz
-   Play Quiz
-   Like / Unlike Quiz
